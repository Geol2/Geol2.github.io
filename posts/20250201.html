Strang의 Linear Algebra를 공부하고 있고 선형대수학은 데이터처리와 컴퓨터 데이터 처리를 다루는 것에 더 적합하다.

또 다르게 미적분학이 항상 언급되는데 말그대로 물리학, 공학, 자연적인 변화와 움직임을 표현하는데에 적합하다. 상호보완적이다.

가우시안 소거법이 가장 먼저 언급되는 것 같고 처음 들어보는 내용이였다.

가우시안 소거법은 행렬을 통해서 연립방정식을 풀어내는 방법이다. 일반적으로 배웠던 소거법이 가우시안 소거법을 간소화한 것이고 많은 데이터를 다뤄야할 땐 행렬을 이용한 방법이 효율적이다.

가우시안 소거법에서 A=LU 분해를 이용하면 Ax=b형태의 연립방정식을 풀어볼 수 있다

A = L U

L : 하삼각행렬, U: 상삼각행렬

A = ( 1 2 4 5 )

해당 행렬을 Factorization 하면 다음과 같다.

( 1 2 4 5 ) = ( 1 0 4 1 ) ( 1 2 0 -3 ) = L U

위처럼 Factorization 과정이 독특했는데 U(상삼각행렬)의 2행 1열을 0으로 강제로 만들기 위해서 U의 2행 1열, 2열을 아래처럼 계산해줘야한다

a 1 1 * -4 * a 2 1 = 0 , a 1 2 * -4 * a 2 2 = -3

위 식을 통해서 (L의 역행렬) * A = U 라는 것을 도출해낼 수 있고 A = L * U가 된다.

그럼 Ly = b(전방대입), Ux = y(후방대입) 순서대로 x = -1, y = -2 값을 도출해낼 수 있다.
